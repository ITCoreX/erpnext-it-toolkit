# ERPNext IT Toolkit

An open-source ERPNext customization project designed to help organizations streamline IT operations, automation, and reporting.

## 🚀 Key Features
- IT Service Request & Ticket Management System
- Network Monitoring Integration (Ping-based alerts)
- Automated Email Notification System
- Monthly Reporting & Dashboard Analytics
- Document & PDF Processing Tools

## 🧠 Tech Stack
- ERPNext (Frappe Framework)
- Python / JavaScript
- MariaDB

## 🏗️ Architecture
This project follows a modular approach using multiple custom Frappe apps:
- IT Service Management App
- Monitoring & Alerts App
- Reporting & Analytics App
- Document Processing App

## 🎯 Objective
To build a scalable and reusable ERPNext toolkit that simplifies implementation for organizations.

## 📌 Current Status
🚧 Initial development phase (actively building core modules)
